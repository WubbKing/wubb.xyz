Setting Up the Marquee Screensaver

To set up the Wubbsoft marquee screensaver:

In Windows 3.1 - NT 3.5:

  1. Open the Control Panel.

  2. Open Desktop Properties.

  3. Set the screensaver to Marquee and click Setup.

  4. Set position to Random and background color to blue.

  5. Enter the text: Wow! Wow! Wubbsoft!

  6. Click Format Text.

  7. Set the font to Hot Coffee 72 point, yellow, and click OK.

  8. Click OK in Marquee Setup to finish.

  9. Click Test to preview your screensaver, then click OK in Desktop Properties to set your screensaver.

In Windows 95 - XP with classic control panel view:

  1. Open the Control Panel.

  2. Open Display Properties and go to Screen Saver.

  3. Set the screensaver to Marquee Display and click Settings.

  4. Set position to Random, speed to 15%, and background color to blue.

  5. Enter the text: Wow! Wow! Wubbsoft!

  6. Click Format Text.

  7. Set the font to Hot Coffee 72 point, yellow, and click OK.

  8. Click OK in Marquee Setup to finish.

  9. Click Preview to preview your screensaver, then click OK in Desktop Properties to set your screensaver.

In Windows XP with category control panel view:

  1. Open the Control Panel.

  2. Choose the Appearance and Themes category and select "Choose a screen saver".

  3. Set the screensaver to Marquee Display and click Settings.

  4. Set position to Random, speed to 15%, and background color to blue.

  5. Enter the text: Wow! Wow! Wubbsoft!

  6. Click Format Text.

  7. Set the font to Hot Coffee 72 point, yellow, and click OK.

  8. Click OK in Marquee Setup to finish.

  9. Click Preview to preview your screensaver, then click OK in Desktop Properties to set your screensaver.
